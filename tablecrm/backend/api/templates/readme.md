Состав шаблона:

Шаблон представляет из себя текстовый файл, который содержит
HTML разметку документа, а так же переменные, циклы, условия,
фильтры, методы. 

Создание шаблона: необходимо указать токен, имя, описание, ID пользователя,
ID типа формируемого документа (поле связано по ключу с type_template).
Тело шаблона прикрепляется к запросу файлом (описание файла выше)
